# Barcode-Engineer-Competence-Std
本仓库聚焦《条码师岗位能力要求》（T/CABC 18.1-2025）团体标准的全生命周期管理，涵盖标准研制、发布与推广全流程核心信息。该标准由中国条码技术与应用协会归口，广西职业技术学院联合中物编码中心、老挝全国工商联等 30 余家单位起草，将于 2026 年 1 月 1 日正式实施。 标准首次将条码师岗位能力划分为初、中、高三个等级，明确各等级资格条件及 “理论知识 + 实践操作” 双维度专业能力要求，填补了我国条码专业人才能力评价标准的空白。仓库收录了标准研制阶段的修改意见采纳记录、技术审查资料，以及发布落地、跨境推广（老挝同步推行）等关键信息，可为条码行业人才选拔、培养、评价提供标准化依据，也为跨境条码人才体系建设提供参考。
